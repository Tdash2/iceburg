# Iceburg
Live Production Control Web Interface. Iceburg is a php based web GUI for controlling broudcast hardware. 

Iceburg is an acronym that stands for 
I ntegrated
C ross
E cosystem 
U ser defined 
R elationship based
G raphical user interface

This acronym explanes what Iceburg does. It is a cross product control interface. 


Below is a list of currently supported devices:
+ Behringer X32, X32 Compact, X32 Rack and X32 Producer
+ Blackmagic Design Videohub
+ Blackmagic Design Videohub Smart Control (40 button)
+ Blackmagic Design Videohub Smart Control Pro (48 button)
+ Cobalt Digital TOPAZ 9905-MPx
+ Blackmagic Design SmartScope Duo
+ VMIX (Tally Only)
+ Symetrix DSP (logic Inputs Only)
+ Tally Iceberg IO Devices (GIP, GPO and TSL 3.0)
